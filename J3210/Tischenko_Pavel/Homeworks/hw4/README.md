# ДЗ4 — SVG-спрайт



- Создан SVG-спрайт `lab2/client/assets/sprite.svg` (иконки: sun, moon, user, logout, search).
- Иконки подключаются через `<use href="assets/sprite.svg#icon-…">`.
- Пример использования: кнопка темы и кнопка выхода в навбаре.



SVG-спрайт нужен, чтобы:

- Переиспользовать иконки без дублирования SVG-кода.
- Держать единый набор иконок и стиль (цвет через `currentColor`, размеры через CSS).
- Уменьшать количество ручной верстки и упрощать поддержку.
