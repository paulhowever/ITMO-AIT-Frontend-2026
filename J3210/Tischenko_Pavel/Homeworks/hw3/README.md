# ДЗ3 — CSS-переменные и темизация



- В `lab2/client/styles.css` введены CSS-переменные (`--bg`, `--surface`, `--text`, …).
- Реализованы 2 темы: светлая (по умолчанию) и тёмная через `html[data-theme="dark"]`.
- Переключатель темы в навбаре (`#themeToggle`) + сохранение выбора в `localStorage`.
- Если пользователь тему не выбирал, используется `prefers-color-scheme`.
